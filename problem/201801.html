# 箭头函数自动执行解决编译后的js只执行一次

项目：《钱隆归来》
时间：2018年1月10日
问题：在引入的级联菜单对话框中，将对话框中显示的数据以外部js文件方式引入，但是当页面后退再返回（重新选择数据条目）后数据列表的条目并不会发生改变
描述：编译后的js脚本只自动执行一次，刷新页面才会重新执行
解决办法：将js文件中的方法改成箭头函数自动执行
问题代码：
……
const cascadeData = ……
export default cascadeData

正确代码:
let formatData = () => {
  const cascadeData = ……
  return cascadeData
}
export default formatData
